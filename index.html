Key Concepts

Clique = every pair of chosen vertices is connected.
Independent Set = no chosen vertices are adjacent.
Complement Graph flips edges/non-edges.

Worked Example

If vertices {0,1,2} all connect to each other, then they form a clique of size 3.

NP-Completeness

Clique is in NP because checking all pairs of a proposed set takes polynomial time.
Reduction: Independent Set → Clique
Construct complement graph G′. Independent Set in G iff Clique in G′.
